# Анализ закономерностей определяющих успешность игр для интернет-магазина.

В данном проекте были выявлены определяющие успешность игры закономерности и потенциально популярный продукт. Был проведен исследовательский анализ данных и проверены две гипотезы о равенстве средних двух генеральных совокупностей методом st.ttest_ind.
