# Analysis of Fitness Center Client Data and Churn Probability Forecast (for the Next Month) for Each Client
Development of a customer interaction strategy for the fitness center based on analytical data.
Building a predictive model to forecast the likelihood of client churn (for the next month) for each individual client and creating typical customer profiles.

As part of the study, an exploratory data analysis was conducted, and a correlation matrix of customer features was created (both as a table and a heatmap).
Additionally, a churn prediction model was built based on customer features. Recommendations were developed from the model to attract more loyal clients.
