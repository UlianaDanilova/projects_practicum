# Анализ данных клиентов фитнес-центра и прогноз вероятности оттока (на уровне следующего месяца) для каждого клиента

Разработка стратегии взаимодействия с клиентами для фитнес-центра на основе аналитических данных. 
Построение модели прогнозирования вероятности оттока (на уровне следующего месяца) для каждого клиента и формирование типичных портретов клиентов.

В ходе исследования был проведен исследовательский анализ данных, составлена матрица корреляции признаков клиентов (в виде таблицы и тепловой карты). 
А также построна модель прогнозирования оттока клиентов на основании их признаков. На основании модели были сформулированы рекампендации для привлечения более лояльных клиентов.
